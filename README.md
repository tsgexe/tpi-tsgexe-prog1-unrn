# La Batalla
> A text-based RPG built in C as a final project for UNRN's Programming 1 course.

This game features randomized stats, tactical turn-based combat against an AI (with attack, defend, heal, and energy-regeneration mechanics), and logging capabilities that save the battle history to a file. 

## Key Features

* **Turn-Based Combat:** Engage in tactical battles against an AI opponent until one participant's health reaches zero.
* **Dynamic Parameters:** Health and energy stats are randomized for both the player and the AI at the start of each match.
* **Action System:** Choose strategically between attacking, defending, healing, or skipping a turn to regenerate energy.
* **Battle Logging:** Automatically exports a turn-by-turn history of the encounter to a local text file.

## Technical Implementation

Developed entirely in C, this project demonstrates practical application of foundational programming concepts:
* Custom data structures (`structs`) for game logic and entities.
* Dynamic memory allocation.
* File handling for data persistence.
